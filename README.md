# RimAI - AI Storyteller for RimWorld

RimAI is an AI-powered storyteller mod for RimWorld that uses advanced language models to create dynamic and intelligent narrative events.

## Features

- **AI-Powered Events**: Uses OpenAI GPT or compatible models to generate context-aware story events
- **Dynamic Storytelling**: Events adapt to your colony's current state, wealth, and recent history
- **Multiple Personalities**: Choose from different AI storyteller personalities (Balanced, Sadistic, Chaotic, Protective)
- **Mod Compatibility**: Intelligent incident resolution works with vanilla and modded incidents
- **Visitor Intelligence**: Advanced visitor observation system with delayed consequences

## Requirements

- RimWorld 1.5+
- 0Harmony
- OpenRouter account (recommended) or OpenAI API key

## Installation

1. Download and install 0Harmony mod
2. Extract RimAI folder to your RimWorld/Mods directory
3. Launch RimWorld and select RimAI storyteller in game settings
4. Configure your API key in mod settings

## Configuration

Access RimAI settings through:
- Main Menu → Mods → RimAI → Configure
- In-game: Settings → Mod Settings → RimAI

### Required Settings:
- **API Provider**: Choose your AI service (OpenRouter recommended)
- **API Key**: Your API key for the selected provider
- **Model**: AI model to use (openai/gpt-oss-120b:free works well)

### Optional Settings:
- **Narrator Personality**: Storyteller behavior style
- **Event Frequency**: How often events occur
- **AI Influence**: Balance between AI and vanilla events

## Supported AI Providers

- OpenRouter (recommended for free tier)
- OpenAI
- Anthropic Claude
- Google Gemini
- NanoGPT
- Groq
- OpenAI Compatible APIs

## How It Works

RimAI analyzes your colony's current state and generates appropriate events using AI. The storyteller considers:

- Colony wealth and population
- Recent events and suffering
- Current season and biome
- Available incident types

Events are generated dynamically and can include custom descriptions, faction assignments, and appropriate difficulty scaling.

## Compatibility

RimAI is designed to work with most RimWorld mods. The intelligent incident resolution system can find and use incidents from other mods automatically.
